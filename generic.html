Converting Raw EEG signal into spectrograms

As a part of research to predict the depth of anestesia using EEG signals and deep learning models, one of the intial task was to extract the signal value from the raw EEG data.the data was converted from .xlsx and .txt file to .jpeg file using MATLAB.During this process as the EEG signals are non-stationary signals and comprises of various unwanted components which were not particularly useful for our analysis were separated from the original signal using Ensemble Empirical Mode Decomposition.

Convolutional Neural Network for Depth of Anesthesia Prediction using EEG images

After converting the raw data of EEG signals into suitable images for CNN analysis,deploying a deep learning model for multiclass classification.The images are classified as Anesthetic Deep, Okay and Low according to their BIS value. The best accuracy is obtained for 10 layer Convolutional layer while decreasing accuracy can be observed for complex models.To see why these discrepency in accuracy occurs refer to the github project in the Learn More Section.
